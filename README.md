# Arrow Puzzle MVP

Mobile-first Arrow Puzzle game built with React + TypeScript + Vite.

## Run

```bash
npm install
npm run dev
```

Open the local URL from Vite (usually `http://localhost:5173`).

## Build

```bash
npm run build
npm run preview
```
